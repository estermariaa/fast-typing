# Fast Typing - Teste de Digitação

Um jogo interativo de teste de velocidade e precisão de digitação. O projeto simula o ambiente de ferramentas como o Monkeytype, oferecendo diferentes níveis de dificuldade e temas visuais.

## Funcionalidades
- **Seleção Dinâmica:** Escolha entre 3 níveis de dificuldade (Fácil, Médio e Difícil) e 3 tempos de partida (15s, 30s e 60s).
- **Feedback em Tempo Real:** As letras mudam de cor instantaneamente (Verde para acerto, Vermelho para erro).
- **Sistema de Ranking:** Resultados salvos localmente (`localStorage`) com filtros por nível e tempo.
- **Temas:** Alternância entre modo Escuro e modo Claro.
